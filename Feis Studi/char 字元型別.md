char 占用的記憶體大小為一個位元
char 利用單引號括住字元 : 'A' 'a' '0' '\n'
printf和scanf的字元格式符 : 使用%c

#include <stdio.h>
#include <stdlib.h>
int main () {
char ch = 'A';            //宣告定義一個叫做 ch 的字元變數,並初始化為 'A' 
printf("%c",ch);          //把大寫 'A'(65) 的編號處存入 ch 
	return 0;
}
